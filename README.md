# Movie Management Scripts

## 使い方

1. 新しくダウンロードした動画ファイルを、Checkinフォルダーに移動する。

2. ディレクトリを移動する。場所は、MOVIE_MNG 直下とする。
例） cd E:\MOVIE_MNG

3. チェックインスクリプトを実行する。
例）python Script\checkin_tool.py

4. プレイリストのデータベースを更新する。
例）python Script\playlist_register.py
